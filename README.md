# Алгоритмический мини-пакет (стрктуры данных и сортировки)
Алгоритмический мини-пакет (стрктуры данных и сортировки), принцип работы: роутинг введенных команд и параметров -> вызов обозначенного мода -> вызов необходимой валидации -> вызов обозначенной функции -> выход из мода по команде -> выход из программы по команде

## Использование
При запуске вас встретит небольшая инструкция, после которой вы можете ввести в терминал необходимую вам команду. Если оно удовлетворило базовые требования к выражению, то вы попадете в мод, в котором можете использовать вызванную вами коману пока не надоест. Если ввод некорректный, вы получите ошибку и ее описание. Большинство ожидаемых ошибок не приведет к завершению программы. Выход из программы осуществляется путем ввода пользователем строк "quit", "q" или "exit", "e".

Команды, которые поддерживаются: 'factorial', 'factorial rec', 'fibo', 'fibo rec', 'bubble sort', 'quick sort', 'counting sort', 'radix sort', 'bucket sort', 'heap sort'.

После попадания в определенный мод, введите последовательность элементов или единственный элемент, в зависимости от задачи. В отдельных модах (например, в bucket sort) необходимо после введения элементов ввести колличество ведер, на что укажет вывод в терминале.

Большинство проверок вынесено в папку validation.
